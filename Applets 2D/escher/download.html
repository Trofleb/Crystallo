Downloading Escher Web Sketch

The instructions below describe how to go about downloading Escher Web Sketch, and an appropriate Java Run-time Environment that you will need to run it. Until operating systems become more java capable out of the box, this isn't as easy of a process as it should be...

The Java Run-time Environment

You will need a copy of either the java run-time environment or a java developer's kit. Instructions for downloading and installing these two products can be found on www.javasoft.com's web pages:

  • The Java Run-time Environment (Windows95/NT, and Solaris).
  • The Java Developer's Kit (Windows95/NT, MacOS, and Solaris).
  • For use on other architectures you will have to find one of the two above packages ported to whatever type of computer you are using.

Downloading Escher Web Sketch

Download Escher Web Sketch here in one of the following compressed formats:

  • escher.zip
  • escher.tar.gz
  • (sadly, I can't provide a stuffit archive)

Running Escher Web Sketch

(You'll need to unpack one of the above two archives first).

  • Unix: Run "jre escher" or "java escher"
  • Win95/WinNT: Run the "escher.bat" script. (You may have to modify the script if you didn't pick a default installation path for the Java Run-time Environment).
  • MacOS: Drag the escher.class file onto the java application included with the java developers kit. (A better MacOS solution is forthcoming)
