Fourier transform

By: Nicolas Schoeni and Gervais Chapuis
Ecole Polytechnique Fédérale de Lausanne, Switzerland
This applet calculates the Fourier transform of a two dimensional density function rho(x) yielding the complex magnitude G(S). The applet is also able to calculate the inverse Fourier transform of G(S). The density function can be either periodic or non-periodic. Numerous tools including drawing tools can be applied in order to understand the role of amplitudes and phases which are of particular importance in diffraction phenomena. As an illustration, the Patterson function of a periodic structure can be simulated.
