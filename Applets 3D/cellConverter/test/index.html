Crystallographic Cell Converter

By: Nicolas Schoeni and Gervais Chapuis
Ecole Polytechnique Fédérale de Lausanne, Switzerland

You need to enable java in your browser

This applet allows to transform the unit cell and its content to another unit cell with contents.
The new cell is specified by a matrix transformation including an origin displacement.
Both input and output files are in CIF format.
